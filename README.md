# Numlang compiler

Numlang is an esoteric stack-based language using only the characters `0123456789^&*+-+/.|;`.

## Features

- Stack-based operations
- Numbers 0-9 push to stack
- + : add
- - : subtract
- * : multiply
- / : divide
- ^ : power
- & : bitwise and
- | : bitwise or
- . : print top of stack
- ; : ignored

## Example

```
23+.
```

Pushes 2, pushes 3, adds (5), prints 5.
